Final project

Reminder: you must cite any source code that you didn't write yourself (see example).

Summary

The basic criteria are: your project must have at least one class of your own creation (aside from the main class), the user must have some way of interacting with objects of that class, and it must use an array. Talk to me in person to get approval for your project idea. Here are some projects from a previous semester if you want ideas.

There is no limit to what you can do. Bonus points are available for features beyond the basic requirements.

Handing it in

If you develop right in Codio, you should be all set. If you are using an external editor, be sure to hand in anything required to compile and run your program (all .java files and any sound/image files).
